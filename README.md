## Installation

Install comfysloth-ecommerce with npm

```bash
  # Clone this repository
  $ git clone https://github.com/dannyberns/react-comfysloth-ecommerce

  # Go into the repository
  $ cd react-comfysloth-ecommerce

  # Install dependencies
  $ npm install

  # Run the app
  $ npm start
```
